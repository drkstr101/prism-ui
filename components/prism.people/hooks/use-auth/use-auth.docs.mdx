A React Hook for managing authentication state. It provides methods for user login, signup, and logout.

## Usage

### Basic Usage

```tsx
const AuthComponent = () => {
  const { user, login, logout } = useAuth();

  const handleLogin = async () => {
    try {
      await login({ username: 'testuser', passwordHash: 'password' });
    } catch (error) {
      console.error('Login failed:', error);
    }
  };

  const handleLogout = async () => {
    logout();
  };

  return (
    <div>
      {user ? (
        <div>
          <p>Welcome, {user.username}!</p>
          <button onClick={handleLogout}>Logout</button>
        </div>
      ) : (
        <button onClick={handleLogin}>Login</button>
      )}
    </div>
  );
};
```

This example demonstrates how to use the `useAuth` hook to manage user login and logout. The `login` function takes a user object with `username` and `passwordHash` properties. The `logout` function clears the user state.

### User Signup

```tsx
const SignupComponent = () => {
  const { user, signup } = useAuth();

  const handleSignup = async () => {
    try {
      await signup({ username: 'test', email: 'test@email.com', passwordHash: 'password' });
    } catch (error) {
      console.error('Signup failed:', error);
    }
  };

  return (
    <div>
      <button onClick={handleSignup}>Signup</button>
    </div>
  );
};
```

This example shows how to use the signup method. The signup function takes user info and sets the user's details in the state.


### Conditional Rendering

```tsx
const ConditionalComponent = () => {
  const { user, logout } = useAuth();

  const handleLogout = async () => {
    logout();
  };

  return (
    <div>
      {user ? (
        <button onClick={handleLogout}>Logout</button>
      ) : <p>Not logged in.</p>}
    </div>
  );
};
```
This example shows how to logout and render components conditionally. If the user is logged in, it shows a logout button; otherwise it shows that the user is not logged in.
